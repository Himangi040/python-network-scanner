# Python Network Scanner

A simple Python-based network scanning tool designed to identify open and closed ports on a target system in a controlled environment.

## Features
- Scans common ports (21, 22, 23, 80, 443)
- Displays open and closed ports
- Logs scan results to a file

## Tools & Technologies
- Python
- Socket Programming
- Linux / Windows

## Usage
1. Clone the repository
2. Run `python scanner.py`
3. Enter a target IP address (e.g., 127.0.0.1)
